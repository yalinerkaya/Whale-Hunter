# Whale-Hunter
- 바이낸스 거래소에서 비트코인 정보를 조회하며 거대한 거래량 포착시 텔레그램에 알림을 보냅니다.
- 분산 서버 환경에서 일어날 수 있는 다양한 이슈를 해결하며 서버의 성능을 개선하고, 유지보수성 높은 프로그램을 작성하기 위해 고민하며 개발하였습니다
